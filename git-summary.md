# Summary about base systax in GIT

1. git init
   digunakan untuk menginisiasi repository dan akan membentuk file .git
2. git merge
   digunakan untuk melakukan penggabungan code dari masing-masing user yang berkontribusi
3. git pull
   digunakan untuk mengambil commit lalu otomatis menggabungkan dengan branch yang aktif
4. git fetch
   digunakan untuk mengambil commit saja tanpa menggabungkan 
5. git status 
   untuk melihat status repository
